# NeoFS API CSharp

### Requirements
- DotNet SDK (3.1.1+)
- protobuf / protoc (3.11.4+)

### Regenerate NeoFS Proto files and documentation
```
# DotNet Restore 
$ dotnet restore

# Regenerate proto files
$ make docgen protoc
```
