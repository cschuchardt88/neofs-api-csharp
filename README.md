# NeoFS API CSharp

[docs](./docs)

[nuget package](https://www.nuget.org/packages/NeoFS.API/)

### Requirements
- DotNet SDK (v5.0.0+)
- protobuf / protoc (3.11.4+)

### Regenerate NeoFS Proto files and documentation
```
# DotNet Restore 
$ dotnet restore

# Regenerate proto files
$ make docgen protoc
```
